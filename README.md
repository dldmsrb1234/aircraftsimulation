# ✈️ 항공기 비행 양상 시뮬레이터

받음각(AoA) · 무게중심(CG) · 양력중심(CP) · 꼬리날개 조건에 따라
항공기 모형이 시간이 지나며 **pitch / roll / yaw** 방향으로 어떻게 변하는지
근사 계산하여 **실시간으로 시각화**하는 교육용 도구입니다.

- 1학기: **항공기 안정성 분석 시뮬레이터**
- 2학기: **모형 항공기 · RC 글라이더 제작 전 설계 비교 도구**

> ⚠️ 이 프로그램은 실제 설계용 CFD가 아니라 **고등학교 과학탐구 및 모형 제작 전 설계 비교용 근사 시뮬레이터**입니다.
> B737 · Cessna · F-15 프리셋은 실제 제조사 공력 데이터가 아니라, 각 유형의 일반적 형상 특징을 반영한 **교육용 근사값**입니다.
> 계산 결과가 실제 비행 성공을 보장하지 않습니다.

   - 🪶 수평꼬리날개: 면적, AoA, 거리, 높이, CL_α
   - 🪁 수직꼬리날개: 면적, 개수, 거리, CL_α
   - 🎯 초기 자세 / 각속도 (pitch·roll·yaw)

- 상단: **전체 안정성 / AoA / CP–CG / Pitch / Roll / Yaw 상태 배지**, **제작 적합성 점수(0~100)**

  ⏹ 정지를 누를 때까지 계속** 시뮬레이션합니다. **⏸ 일시정지 / ⏹ 정지(처음으로) / 속도(0.25~4×)**,
  **마우스 드래그=시점 회전, 휠=확대**. 서버 재실행이 없어 랙 없이 부드럽습니다.
  *(Three.js 를 CDN 에서 불러오므로 첫 실행 시 인터넷 연결이 필요합니다.)*
  *(아래 ‘📈 그래프’·‘🧪 분석’ 탭은 ‘시뮬레이션 시간(s)’ 슬라이더로 정한 분석 구간 기준입니다.)*
- **🛩️ 내 STL 모델 사용 + 항공역학 특성 반영**: 사이드바 *‘3D 모델 (STL 업로드 + 형상 분석)’* 에서
  `.stl` 을 올리면 기본 모델 대신 **그 모델**로 비행 자세가 표시됩니다.
  - **축 정렬**: 모델의 *기수(앞) 축* 과 *위 축* 을 골라 표준 자세로 맞춥니다(3D에서 바로 확인).
  - **STL 단위**(mm/cm/m) 선택 후 **‘📐 STL 형상의 항공역학 특성 적용’** 을 누르면,
    형상에서 **기체 길이·날개폭·높이·주날개 면적·무게중심(CG)·양력중심(CP)·관성모멘트(Ix·Iy·Iz)** 를
    추출해 시뮬레이션 입력에 자동 반영합니다 → **모델 형상이 비행 안정성·동역학에 실제로 반영**됩니다.
  - 추출 방식: CG·관성은 *균일 밀도* 폴리헤드론 적분으로 계산합니다.
  - **표면 패널 공력(ray 기반)**: STL 을 적용하면 단순 요약값이 아니라, **표면 삼각형마다 기류가

    적분하고, 그 결과가 **실제로 pitch·roll·yaw 동역학을 구동**합니다. 즉 형상·자세를 바꾸면
    양력·모멘트가 분포째 바뀝니다.
  - **무게중심(CG)으로 안정 조절**: 모멘트를 원점 기준으로 저장하므로, 사이드바 *무게중심 CG*

  - 애니메이션 **‘모델 조절’** 막대에서 **크기·미세회전(X/Y/Z)** 도 실시간 조절. (이진 STL·폴리곤 적은 모델 권장)
- **⏱️ 시각 t 슬라이더**: 정적 분석용. 원하는 시점을 고정해 아래 Plotly 그림·그래프를 자세히 봅니다
- 탭
  - **🛩️ 비행 자세**: 측면도(pitch) · 정면도(roll) · 상면도(yaw) · 3D 자세
    (반투명=초기 자세, 진한 색=현재 자세, 화살표=양력 벡터·모멘트 방향)

## 5. 물리식 설명

### 5-1. 기본 공식 — **고2 수준 (역학·삼각비 범위)**

**① 동압**

$$q = \tfrac{1}{2}\,\rho\,V^2$$

**② 받음각에 따른 양력계수 (작은 각도, 선형 근사)**

$$C_L = C_{L\alpha}\,\alpha \qquad (\alpha \le \alpha_{stall})$$

실속각을 넘으면 $C_L$이 더 이상 커지지 않고 점차 감소(실속 근사). *내부 계산은 라디안 사용.*

**③ 양력 = 동압 × 면적 × 양력계수**

$$L_{wing}=q\,S_{wing}\,C_{L,wing}, \qquad L_{tail}=q\,S_{tail}\,C_{L,tail}$$

**④ pitch 모멘트 = 힘 × 거리** (지렛대 원리)

$$M_{pitch}=L_{wing}\,(x_{cg}-x_{cp})\;-\;L_{tail}\,l_{tail}$$

- 양력중심(CP)이 무게중심(CG)보다 **앞**이면 → 기수 들림
- CP가 CG보다 **뒤**이면 → 복원(안정)

**⑤ roll 모멘트 = 비대칭 양력 × 가로 거리**

$$M_{roll} = (\text{비대칭}\times L_{wing})\times \tfrac{b}{4}$$

**⑥ yaw 모멘트 = 수직꼬리 측력 × 거리**

$$M_{yaw} = -\,(q\,S_v\,n\,C_{L\alpha,v}\,\psi)\times arm$$

(기수가 돌면 수직꼬리가 되돌리려는 힘 → 방향 안정)

**⑦ 양력중심의 이동(선택, 1차 근사)**

$$x_{cp}=x_{cp,base}+k_{cp}\,\alpha$$

### 5-2. 심화 공식 — **고2 과정에서 벗어나는 부분 (탭 구분)**

> 아래는 “회전 운동방정식 + 수치적분”으로, 미분·관성모멘트 개념이 들어가 고2 정규 교육과정을 넘습니다.
> 보고서에서는 “시간에 따라 자세가 변하는 과정을 컴퓨터로 반복 계산했다” 정도로 서술하면 됩니다.

**⑧ 회전 운동방정식 (감쇠 포함)** — *뉴턴 회전 제2법칙*

$$\text{각가속도}=\frac{M-c\cdot\dot\theta}{I}$$

여기서 $I$=관성모멘트, $c$=감쇠계수.

**⑨ 시간 적분 (전진 오일러법)** — *수치해석*

$$\dot\theta \mathrel{+}= \text{각가속도}\cdot dt, \qquad \theta \mathrel{+}= \dot\theta\cdot dt$$

이를 pitch·roll·yaw 세 축에 대해 매 시간 단계마다 반복합니다.

**⑩ 정적 안정 지표** — *미분 개념*

$$k_\theta=-\frac{dM_{pitch}}{d\theta}\;(>0\Rightarrow \text{세로 안정})$$

## 6. AoA 와 양력중심 개념

- **받음각(AoA, α)**: 날개가 공기 흐름과 이루는 각도. 커질수록 양력↑, 그러나 **실속각**을 넘으면 공기 흐름이 떨어져 양력이 급감(실속).
- **양력중심(CP)**: 날개가 받는 양력의 “대표 작용점”. 받음각이 변하면 위치가 조금씩 이동.
- **무게중심(CG)과의 관계가 세로 안정의 핵심**:
  - CP가 CG **뒤** → 자세가 흐트러지면 스스로 복원(안정)
  - CP가 CG **앞** → 자세가 점점 더 틀어짐(불안정)

## 7. 실제 제작에 활용하는 방법

1. 만들 모형의 크기(길이·날개폭·면적)·무게를 입력하고 **사용자 정의형**에서 시작.
2. **무게중심(CG)을 양력중심(CP)보다 약간 앞**에 두어 `k_θ>0`(세로 안정)이 되게 조정.
3. **수평꼬리 면적·거리**를 늘려 pitch 진동이 빠르게 수렴하는지 확인.
4. **수직꼬리 면적·개수**로 yaw 방향이 안정적으로 수렴하는지 확인.
5. **좌우 비대칭**을 0에 가깝게 → roll·yaw 치우침 최소화(제작 시 좌우 대칭이 중요).
6. **제작 적합성 점수**와 **자동 해석 문장**을 보고서 근거로 활용.
7. 실제 글라이더는 시뮬레이션의 CG 위치를 무게추로 재현해 던져보며 비교.

## 9. 검증 및 경고

- 받음각 **12° 이상 → 실속 주의**, **18° 이상 → 실속 위험** 경고를 표시합니다.
- 선형 양력계수 근사는 **작은 받음각에서만** 유효합니다.
- 양력중심 이동은 실제 익형 해석이 아니라 **단순 근사**입니다.
- 본 도구는 **설계 비교용**이며 실제 항공기 성능 예측이 아닙니다.

## 10. 모델의 한계

- 익형(airfoil) 형상, 압축성, 점성, 후류(downwash) 등을 정밀하게 다루지 않습니다.
- pitch·roll·yaw 축을 **독립**으로 근사하여 축 간 결합(coupling)을 단순화했습니다.
- roll에는 **상반각(dihedral) 복원이 없어** 좌우 비대칭이 있으면 계속 구르는 경향이 나타납니다.
- 관성모멘트·감쇠계수는 자세 변화가 눈에 잘 보이도록 조정한 **교육용 값**입니다.
- **STL 표면 패널 공력의 한계**: 표면 경사(impact/inclination) 기반의 **비점성 근사**라,
  실제 익형의 순환(circulation) 양력을 정밀하게 재현하지는 않습니다(저속 양력 기울기는 실제보다 낮게 추정).
  자기 그림자(occlusion)는 무시하고 면 단위 windward 판정만 사용하므로 오목 형상은 과대평가될 수 있습니다.
  무게중심·관성은 *균일 밀도* 가정이라 실제 무게 분포(엔진·배터리 등)와 다를 수 있습니다.
  STL 단위(mm/cm/m)를 잘못 고르면 면적·관성이 크게 어긋나므로 적용 후 *길이* 값으로 단위를 꼭 확인하세요.
- 균일밀도 무게중심이 양력중심보다 **뒤**면 형상이 정적으로 불안정해 비행 자세가 **발산**합니다
  (실제 모형도 뒤집힘). 이때는 경고가 뜨며, **무게중심(CG)을 앞으로**(기수 무게추) 옮기면 안정화됩니다.
  STL 적용 시에는 안정성 판단을 명확히 하기 위해 *AoA에 따른 CP 자동 이동*을 끕니다.
- 따라서 결과의 **절대 수치보다 “경향과 비교”**에 의미를 두어야 합니다.
